Position Information

We have a 4 types of the positions.

  • Position - Sticky

    When we use this then content is always set with the vieport . it will stick with body . we have to set the value
    like top -0 to sticky at that place

  • position-fixed

    When we make the fixed the element will stick with page it will not move any where
    here also we have to set the value.

  • position-relative

    When we make the fixed the element will stick with page it will not move any where
    here also we have to set the value.

  • position-absotute

    When we make the fixed the element will stick with page it will not move any where
